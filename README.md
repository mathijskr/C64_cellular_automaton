# C64_cellular_automaton

2D Elementary cellular automaton.

The program can be assembled by running java -jar **PATH_TO_KICKASS**/KickAss.jar cell.asm.
To run the program in VICE:
1. x64 cell.prg.
2. POKE 2049, **RULE**
3. SYS 4096
